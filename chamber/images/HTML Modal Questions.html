Question 2

Which of the following would set the background color for only the input elements on an HTML document that have the text type?
✅ Correct Answer(s):

  • input[type=text] {background-color: #efac3e;}

Explanation:

  • input {background-color: #efac3e;} would apply the background color to all <input> elements, not just text inputs.
  • input[name=text] would only match <input> elements with name="text", which is incorrect for selecting based on type.
  • text {background-color: #efac3e;} is invalid since there is no HTML <text> element.

Question 3

How do you invoke a callback function in JavaScript?
✅ Correct Answer:

  • By passing it as an argument to another function and calling it later

Explanation:
A callback function is a function passed as an argument to another function, which later executes it. Example:

js
function greet(name, callback) {
    console.log("Hello, " + name);
    callback();
}

greet("Pius", function() {
    console.log("Welcome!");
});

  • Handling asynchronous operations, such as fetching data from a server

  • Fetching data from an API
  • Handling events
  • Running functions after animations

✅ Correct Answer:

js
localStorage.setItem("user", JSON.stringify({ name: "Pius" }));

To retrieve:

js
let user = JSON.parse(localStorage.getItem("user"));
